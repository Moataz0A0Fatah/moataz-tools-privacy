    <h1>Privacy Policy for Moataz's Tools</h1>
    <p><strong>Last updated:</strong> March 2026</p>

    <h2>1. No Data Collection</h2>
    <p>Moataz's Tools (the "Add‑in") does <strong>not</strong> collect, store, or transmit any personal data. It operates entirely offline, with no internet connection required.</p>

    <h2>2. Local File Access</h2>
    <p>The Add‑in only reads files from your local computer (e.g., <code>.rfa</code> and <code>.rvt</code> files) in folders you specify. It generates local XML catalogues for internal use. No files are uploaded or shared externally.</p>

    <h2>3. Third‑Party Sharing</h2>
    <p>Because the Add‑in collects no data, it does not share any data with third parties.</p>

    <h2>4. Data Retention & Deletion</h2>
    <p>The Add‑in does not retain any personal data. All generated files (catalogues) are stored locally and can be deleted manually by the user at any time.</p>

    <h2>5. User Consent & Deletion Requests</h2>
    <p>Since no personal data is collected, there is no need to revoke consent or request deletion. If you have any concerns, please contact the developer.</p>
